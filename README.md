# KSP-Auto-landing
*Some python scripts enable your Kerbal rocket to land (safely, hopefully) at a fixed point*

---

##### CAUTION: 
This project is still **IN EARLY STAGE OF DEVELOPMENT**, therefore some functions may not yet completed and buggy.

---

##### Usage:
To run the code:
`python landing.py`
